Principais formatações

Negrito/Destaque

Para inserir um texto em negrito como ESTE, insira a tag <strong> (tag de semântica) ou
ESSE pela tag <b> (com o mesmo objetivo, porém só sua com forma).

Itálico/

Já se o seu objetivo for formatar um texto em itálico, utilizamos da tag <i>(não semântica) tag <em>

!!!!OBSERVE!!!!

(OBS: Como o HTML é uma linguagem de marcação por semântica, é recomendavél utilizar da tag <strong> e a tag <em>).

Texto marcado

Podemos marcar um texto com a tag <MARK> (pouco usual- podendo mudar a cor, por exemplo, apenas pelo CSS).

E no outro parágrafo, temos outro texto marcado no final.

Dica: para selecionar que todas as marcações contenham a mesma cor, deve-se abrir uma tag no <HEAD> chama <style>,
e colocar váriavel 'mark' com o 'background-color' da cor que desejar.

Texto GRANDE e pequeno

Estamos criando um texto grande e um texto pequeno nesse parágrafo.
(OBS:Como a tag <small>; cedê a um significado, ela ainda é utilizada e validada,
já a tag <big> é obsoleta, por mais que ainda esteja funcionando.)

Texto deletado

Podemos marcar um texto como 'excluido' para indicar que ele deve ser lido,
mas não considerado com a tag <del> .

Texto inserido

Podemos marcar um texto como 'inserido' para expressar a ideia que ele foi adicionado depois com a tag <ins>.

Existe também o sublinhado com a tag U (não semântica-apenas de forma).

Texto sobrescrito

Para inserir coisas do tipo: x20+3 adicione a tag <sup> (Em cima ↑).

Texto subescrito

Para inserir coisas do tipo: H20 adicione a tag <sub> (Em baixo ↓)
