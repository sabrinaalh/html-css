Usando Links Externos

O que são links externos?
São links para outros sites (fora os nossos).

Você pode acessar o PDF dessa aula no repositório público no Github.

Você também pode acessar o canal gratuito no YT do Gustavo Guanabara.

Para com que o link fosse contido no texto, ele foi envelopado e inserido na tag <a href="http://www.algumacoisa.com">Texto excolhido</a>

(DICA:Esse link vai para uma nova navegação, e provavelmente ele nunca mais irá voltar.
Então, quando o link for externo, é recomendado usar dentro da tag A 'target="_blank" e 'rel="external".
Veja a diferença entre o Link do Github [sem] e o do Youtube [com])
